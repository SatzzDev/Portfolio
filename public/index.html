Fullstack Developer

Saya seorang pengembang fullstack dengan pengalaman lebih dari 5 tahun dalam membangun aplikasi web modern dari frontend hingga backend. Spesialisasi saya mencakup React, Next.js untuk frontend dan Node.js, Express untuk backend.

Di luar pengembangan perangkat lunak, saya menikmati eksplorasi teknologi baru, kontribusi open source, dan berbagi pengetahuan melalui blog dan konferensi.
